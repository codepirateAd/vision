Vision


Vision is a web-based learning platform where students across the globe can learn the computer vision library i.e. OpenCV. Vision platform contains all the information about computer vision and OpenCV.The Webpages contains the code for the particular branch of Computer Vision which can be tested directly on the website without the need of copying the entire code to a local system.
