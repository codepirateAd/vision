Bitwise Operations

Bitwise operations are used in image manipulation and used for extracting essential parts in the image. In this section, we will be learning the following bitwise operations :
  1. AND
  2. OR
  3. XOR
  4. NOT

import cv2
import numpy as np

img1 = np.zeros((250, 499, 3), np.uint8)
img1 = cv2.rectangle(img1,(200, 0), (300, 100), (255, 255, 255), -1)
img2 = cv2.imread("image.png")
bitAnd = cv2.bitwise_and(img2, img1)

cv2.imshow('Bitwise And', bitAnd)

cv2.waitKey(0)
cv2.destroyAllWindows()
