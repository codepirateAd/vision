Simple Thresholding

In this tutorial, you will learn how to use OpenCV and the cv2.threshold function to apply basic thresholding and Otsu thresholding.

Thresholding is one of the most common (and basic) segmentation techniques in computer vision and it allows us to separate the foreground (i.e., the objects that we are interested in) from the background of the image



Thresholding Binary

In Binary threshold we are comparing each pixel of original pixel to 50. If the value of the pixel is less than 50 the value is assigned to 0 and if the value is greater than 50 then the pixel value is assigned 255.

import cv2 as cv
import numpy as np

img = cv.imread('gradient.jpg',0)
                    # (source, threshold_value, maximum_value, thresh_type)
_, th1 = cv.threshold(img, 100, 255, cv.THRESH_BINARY)

cv.imshow("Image", img)
cv.imshow("th1", th1)

cv.waitKey(0)
cv.destroyAllWindows()
